# 富士现场配方师 / Fujifilm Scene Recipe Master

根据用户上传的现场图，分析光线、反差、色温、主体和目标风格，生成一套可以直接设置到富士相机中的 JPEG 直出色彩配方。

Analyze an uploaded scene photo and produce one practical Fujifilm JPEG recipe matched to the light, contrast, color temperature, subject, camera model, and target look.

## 能做什么 / What It Produces

| 输出 | 说明 |
|---|---|
| 现场判断 | 判断光线、反差、色温、主体、主要风险与置信度。 |
| 富士直出配方 | 给出胶片模拟、动态范围、白平衡偏移、高光、阴影、色彩、锐度、颗粒等可执行参数。 |
| 机型兼容处理 | 用户提供机型时，只输出该机型支持的参数；不兼容项会删除或给出接近的折中方案。 |
| 现场微调 | 试拍后只检查曝光、色偏和暗部三类问题，避免信息过载。 |
| 一句话抄机版 | 把完整参数压缩成一行，方便手机查看、复制和保存。 |

## 支持的输入 / Supported Inputs

- 只上传现场图。
- 现场图 + 相机型号，例如 X100VI、X-T5、X-S20、X100V、X-E4、X-T30 II。
- 现场图 + 目标风格，例如日系清透、胶片感、电影感、港风、低饱和、暖调人像、蓝调夜景。
- 现场图 + 相机型号 + 目标风格。
- 没有图片时，也可以只描述场景，但会降低判断置信度。

## 特点 / Features

- **按现场判断**：不是机械套用固定网红配方，而是先判断真实光线和环境色。
- **直接可执行**：默认只给一套主配方，并优先输出能直接抄进相机的参数。
- **机型感知**：避免输出用户机型不支持的胶片模拟或菜单参数。
- **动态范围联动**：同时考虑动态范围、最低 ISO、动态范围优先以及高光/阴影设置之间的关系。
- **彩色与黑白分开**：彩色配方和 ACROS 等黑白配方使用不同参数模板。
- **适合手机现场使用**：输出保持简洁，并提供一句话抄机版。
- **避免虚假确定性**：截图、滤镜、HDR、平台压缩和屏幕显示都会影响判断，配方被明确定位为现场试拍起点。

## 使用方式 / How To Use

1. 将整个 `FlyWolf-fuji-scene-recipe-master` 文件夹复制到支持 Skill 的 Agent 技能目录。
2. 上传一张现场图。
3. 直接说：

```text
根据这张现场图给我一套富士直出参数。
```

更完整的输入示例：

```text
这是现场图，我用富士 X100VI，想要低饱和电影感。给我一套可以直接抄进相机的参数。
```

也可以只描述场景：

```text
阴天咖啡馆窗边人像，用 X-T5，想要暖调但肤色不要黄。
```

## 默认输出结构

1. 现场结论：光线、反差、色温风险和判断置信度。
2. 推荐配方：一套完整参数表。
3. 试拍后微调：只说明最相关的曝光、色偏和暗部调整。
4. 一句话抄机版：完整参数压缩为一行。

必要时只增加一套简化备选，不重复输出第二张完整表格。

## 文件说明 / Notes

- `SKILL.md`：供 Agent 读取的完整判断逻辑、参数规则、输出模板和禁止事项。
- `README.md`：供用户阅读的用途、输入方式、使用示例与目录说明。
- `agents/openai.yaml`：可选的 Agent 界面元数据和默认调用提示。
- 该 Skill 不包含脚本、模型文件、密钥或运行时依赖。
